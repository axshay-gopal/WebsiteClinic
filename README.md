# 🦷 Pranam Dental Speciality Care Centre

A professional, minimal, and high-performance website for **Pranam Dental**, located in Kannur, Kerala. Built with a modern "Medical Orange" theme, this site is designed to convert visitors into patients via a seamless WhatsApp-integrated booking system.

## ✨ Features

- **Orange Medical Theme:** A warm, professional color palette designed for the healthcare industry.
- **Serverless Appointment System:** Patients book appointments through a modal that redirects to the clinic's WhatsApp with pre-filled details.
- **Fully Responsive:** Beautifully optimized for Desktop, Tablet, and Mobile screens.
- **Zero Backend Required:** Uses the WhatsApp Business API to handle incoming leads directly to the clinic's phone.
- **Fast Performance:** Optimized with Vite for near-instant load times and high SEO scores.

---

## 🛠️ Tech Stack

- **Frontend:** [React.js](https://reactjs.org/) (Functional Components & Hooks)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Build Tool:** [Vite](https://vitejs.dev/)
